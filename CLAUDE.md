# North Star

An immersive, cozy, top-down web game inspired by *Animal Crossing*, designed as a playable interactive CV.

# Tech

## JS

Uses Kaplay.js to be easily distributed in web and fully developed by an AI agent.
When working on kaplay code, use the kaplayjs-docs skill.

## Assets

User provided raw assets under public/assets folder.
The idea is to have a first shot that the user can refine when splitting sprites.
Use 2d-tileset-assets skill when needed to help the user.
